# AssekTrust - Plateforme de Certification d'Entreprises 🛡️

Combattez les arnaques en ligne au Gabon! AssekTrust est une Progressive Web App (PWA) qui permet aux entreprises de se faire certifier et aux clients de vérifier leur authenticité avant d'acheter sur les réseaux sociaux.

## ✨ Fonctionnalités Principales

### Pour les Entreprises
- 📝 S'inscrire avec un compte personnel
- 💳 Choisir un plan d'abonnement (Mensuel, Trimestriel, Premium)
- 💰 Payer via Mobile Money (Airtel Money, Moov Money)
- 🎫 Obtenir un badge de certification imprimable
- 📱 Afficher le badge sur les réseaux sociaux
- 📊 Suivre l'état de leur certification

### Pour les Clients
- 🔍 Rechercher et vérifier les entreprises certifiées
- ✓ Consulter les détails des entreprises vérifiées
- 🌐 Voir les réseaux sociaux de l'entreprise
- 👁️ Distinguer les entreprises légitimes des arnaqueurs

### Pour l'Administrateur (denjidem243@gmail.com)
- 🔐 Connexion sécurisée avec mot de passe
- ✅ Valider manuellement les entreprises
- 💰 Confirmer les paiements reçus
- 🎨 Personnaliser l'application
- 📊 Dashboard avec statistiques en temps réel

3. Cliquer sur "Deploy"
